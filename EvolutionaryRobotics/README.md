This project implements a GeneticAlgorithm to demonstrate below topics.
1) Encoding comparison: Encoding 1 vs Encoding 2.
2) Constraint handling comparison: Penalty fitness vs Repair-based fitness.
3) Operator ablation:
• mutation-only (no crossover),
• crossover-only (no mutation),
• both (standard EA).
4) Selection comparison:  tournament with two different sizes ( 2 vs 5)).
